mbed_xdocker sample app
=======================

Prepare
-------
1. build container (read ../../README.md)
2. prepare mbxc script

   ```
   docker run mbed > mbxc
   chmod +x mbxc
   ```

Build
-----
1. start make

    ```
    ./mbxc make
    ```

Clean
-----
1. start make clean

    ```
    ./mbxc make clean
    ```
